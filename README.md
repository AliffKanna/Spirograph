Reliving Spirograph memories by coding parametric curves! Features real-time drawing with **turtle** and smooth rainbow colors using **colorsys**.

## Features
- Real-time line-by-line drawing with turtle
- Rainbow color transitions using colorsys
- Dual and layered designs
- Easy to tweak scales, colors, and equations
